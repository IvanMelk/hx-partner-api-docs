QuoteResponse

Representation of a quote reflecting it's current contents.

Property Name Type Description
QuoteId string The quote's ID.
Package_Data Package The main package's details.
Voyage_Data Voyage The main voyage's details.
Supplemental_Data Supplemental Supplemental packages.
Passengers Passenger[] The passengers assigned to the quote.
PaymentInformation_Data PaymentInformation Payment information for the quote.
