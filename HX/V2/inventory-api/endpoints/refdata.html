Get Refdata Files

METHOD: GET

Endpoint: /refdata

Query Parameters

Name Required JSON Type Remarks
Locale No String The locale to serve the content in.

Example

Request

curl --location '{:apiBase}/refdata?local=en-US'

Response

Type: File Download

Contents:

  • excursions_{locale}.json
  • portlist.json
  • ships_{locale}.json
  • voyages_{locale}.json
