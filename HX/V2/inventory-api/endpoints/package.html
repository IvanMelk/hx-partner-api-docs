Get Package Files

METHOD: GET

Endpoint: /packages

Query Parameters

Name Required JSON Type Remarks
Market No String The market within which to search.
Currency No String The ISO 4217 conformant currency code the packages are to be sold in.
Method No String Return either a ZIP archive download or as JSON in the response body. Possible options: archive, json. Defaults to json.
Skip No Number The amount of items to offset the search results by. Only used in JSON results.
Take No Number The amount of items to include in the returned results. Only used in JSON results.

Example

Request

curl --location '{:apiBase}/packages?market=no&currency=sek'

Response

{
    "total": 1234,
    "returned": 20,
    "data": [
        // ommitted, see note
    ]
}

note: For details about the package payload, see Package.

Request

curl --location '{:apiBase}/packages?market=no&currency=sek&method=archive'

Response

Type: File Download

Contents:

  • {market}_{currency}_packages.json
