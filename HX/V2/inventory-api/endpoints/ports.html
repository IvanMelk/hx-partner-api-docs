Get Ports

METHOD: GET

Endpoint: /refdata/ports

Query Parameters

Name Required JSON Type Remarks
Skip No Number The amount of items to offset the search results by.
Take No Number The amount of items to include in the returned results.

Example

Request

curl --location '{:apiBase}/refdata/ports'

Response

{
    "total": 860,
    "returned": 20,
    "data": [
        // ommitted, see note
    ]
}

note: For details about the port payload, see PortDetails.
