Extras

Extras provide enhancements to a package voyage in the form of addons, excursions and additional meals. While a package may include a base selection of extras, it is also possible to add more for an additional cost.

Searching for extras

You can search for extras through the Search Extras endpoint.

Adding an extra to a quote

To add an extra to quote, you currently need to supply the extra id and passenger selection through the Add Extras endpoint.

Removing an extra from a quote

To remove an extra from a quote, you currently need to supply the extra id and passenger selection through the Remove Extras endpoint.
